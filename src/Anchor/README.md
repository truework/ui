# Anchor

```js
import { A } from '@truework/ui';
```

## Usage

As an anchor `<a>`:

```jsx
<A href="/">Click Me</A>
```

As a `react-router` `Link`:

```jsx
import { Link } from 'react-router';

<A as={Link} to="/">
  Click Me
</A>;
```
