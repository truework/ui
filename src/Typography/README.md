# Typography

```js
import { H1, H2, H3, H4, H5, H6, P } from '@truework/ui';
```

## Usage

```jsx
<H1 mb='med' color='primary'>
  Hello world!
</H1>
```

### Options

All typography primitives support `typography`, `space`, `layout`, and `color`
props from `styled-system`.
